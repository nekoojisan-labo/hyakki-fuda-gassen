# 百鬼札合戦

和風妖怪を題材にした、1人用ブラウザカードゲームの試作版です。外部通信やアカウント登録はありません。

公開版: [百鬼札合戦をブラウザで遊ぶ](https://nekoojisan-labo.github.io/hyakki-fuda-gassen/)

## 起動

1. このフォルダで `npm start` を実行します。
2. Chrome または Safari で `http://127.0.0.1:4173` を開きます。
3. 「新しい試合」を押します。中断した試合があれば、ターン開始時点から再開できます。

初回はルール画面が自動で開きます。試合中は右上の「？ ルール」からいつでも再表示できます。「ヒント ON」では、次に押せるカード・盤面枠・ボタンが金色に光ります。

カードを選ぶと「条件・対象・結果」を表示します。攻撃は、自分のモンスターと相手の対象を選び、戦闘予測を確認してから「攻撃実行」で確定します。召喚、効果、罠、ダメージ、回復は中央へ順番に表示され、「直近の出来事」から読み返せます。

## 配置確認

画面右上の「座標」ボタンを押すと、カード枠の名前・中心点・当たり判定が表示されます。PCとスマートフォンで同じカードが正しい枠へ入っているかを、コードを読まずに確認できます。

## 検証

`npm run check` でカード、画像参照、PC／スマホ座標、効果処理、戦闘予測、ゲームルールをまとめて確認します。

## 公開

`master` への更新はGitHub ActionsからGitHub Pagesへ静的配信します。対戦処理と保存はブラウザ内だけで動き、公開版でも外部APIへ対戦データを送りません。
